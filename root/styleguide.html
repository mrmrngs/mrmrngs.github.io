All Page Headings

Heading Two

Heading Three

All h3 headings are used for headings within sections(standard content within a page/document flow), and should be in Elido, sans-serif; font size 2.4em

Heading Four

All h4 headings are for pull quotes, italic; font size 2.6em

Heading Five

All h5 headings are for form questions, and should be in Darker Grotesque, font size 1.8em, font weight 500

Heading Six

All h6 headings are used for decorative labels, and should be in Kalam, color #5f8c56, either 1em or 1.4rem depending on label function
